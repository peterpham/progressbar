Render progress bars and a set of controls.
This project is built with
* Gulp
* ES6/Babel
* RequireJS
* QUnit
* SASS/SCSS
* Nunjucks

### Getting started

```
$ npm install
$ gulp
```

### Demo
[project root]/index.html

### Unit Testing (QUnit)

Via Browser:

[project root]/tests/testrunner.html

Via Gulp/PhantomJS:

```
$ gulp test
```
